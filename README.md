# awsscrape
This Python script will parse Data from ec2instances.info 

requires lxml, requests 

Usage: python parse_ec2instances_info.py apiname object

Example: python parse.py t2.micro memory 

Valid objects are  

name  

apiname 

memory 

computeunits 

vcpus 

ecu-per-vcpu 

storage 

architecture 

networkperf 

ebs-max-bandwith  

ebs-throughput 

ebs-iops 

maxips 

enhanced-networking 

vpc-only 

linux-virtualization
